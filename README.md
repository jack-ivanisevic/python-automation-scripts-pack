# Python Automation Scripts
## Overview

This repository contains a curated collection of Python automation scripts designed to streamline data tasks, support CRM workflows, process JSON structures, interact with APIs, and assist with common technical operations. These scripts are examples of practical, real-world automation patterns that are lightweight, modular, and easy to understand.

The purpose of this repository is to demonstrate applied automation skills in a way that is platform-agnostic and suitable for CRM, technical marketing, engineering, or operations environments.

---

## Types of Automations Included

### Data Cleaning and Transformation
Examples of small scripts that:
- Normalize CSV data  
- Clean inconsistent fields  
- Sanitize email lists  
- Remove duplicates  
- Standardize column structures  

### JSON and API Utilities
Scripts demonstrating:
- JSON parsing and restructuring  
- API request handling  
- Error-safe retry logic  
- Payload transformation for CRM or automation platforms  

### CRM-Related Utilities
Non-proprietary examples that:
- Prepare lists and segments  
- Validate field consistency  
- Convert data into profile-friendly formats  
- Apply formatting rules for automations  

### File and System Automations
Lightweight utilities such as:
- File renaming tools  
- Folder watchers  
- Basic scheduling examples  
- Helpers for logging and monitoring
